O RPG System é um sistema desenvolvido em Python com interface gráfica utilizando Tkinter e banco de dados SQLite, criado para auxiliar na organização de campanhas de RPG de mesa. O projeto permite o gerenciamento completo de personagens, NPCs, campanhas e sessões, oferecendo uma experiência prática e intuitiva para jogadores e mestres.

O sistema conta com funcionalidades de criação, visualização, alteração e exclusão de personagens, incluindo diferentes classes como mago, guerreiro, ladino, arqueiro e clérigo. Além disso, é possível cadastrar NPCs com suas respectivas funções dentro da campanha.

Na área de campanhas, o usuário pode criar histórias completas, visualizar detalhes das aventuras e organizar sessões vinculadas a cada campanha. O sistema também permite armazenar descrições detalhadas das sessões, facilitando o acompanhamento do progresso das aventuras.

O projeto foi desenvolvido com foco na prática de conceitos fundamentais e intermediários da programação em Python, como funções, estruturas condicionais, loops, interface gráfica com Tkinter, manipulação de banco de dados com SQLite, organização de código, eventos de interface e interação dinâmica com o usuário.
